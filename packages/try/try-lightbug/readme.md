#     

## Quickstart

- install:

```ruby

# add mojo package
task t:tl:pm -- add lightbug_http

# install mojo package
task t:tl:pm -- install
 
 
```

- quick run:

```ruby

# magic run mojo src/try_01.mojo 
task t:tl:dev    

 
```

- run with `magic shell`

```bash


# activate mojo virtualenv
task t:tl:venv

# run http server: 
task t:tl:run

# deactivate mojo virtualenv
exit

```

- http web: http://127.0.0.1:8080
